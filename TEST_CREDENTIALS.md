# Test Account Setup

## Quick Setup

1. **Open the setup page**: Open `create-account.html` in your browser
2. **Fill in the form**:
   - Name: Your name (e.g., "Admin User")
   - Email: Your email (e.g., "admin@dholakia.com")
   - Password: Choose a password (minimum 6 characters)
3. **Click "Create Account"**
4. **Save the credentials** shown on the page
5. **Go to `index.html`** and log in with those credentials

## Alternative: Use Signup Page

You can also use the built-in signup functionality:
1. Go to `index.html`
2. Click "Sign up" link on the login page
3. Create your account there

## Default Test Credentials (if you want to create manually)

If you want to create a test account with these specific credentials:

- **Email**: `admin@dholakia.com`
- **Password**: `admin123` (or any password you choose)

Just use the `create-account.html` page with these values.

## Notes

- The first account created will have "Admin" role by default
- Make sure your Supabase project has authentication enabled
- Email confirmation may be required depending on your Supabase settings
- If email confirmation is enabled, check your email after signup
